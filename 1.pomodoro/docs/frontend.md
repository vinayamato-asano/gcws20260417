# フロントエンド構成・UI仕様

このドキュメントは、ポモドーロタイマーWebアプリのフロントエンド構成と主要UI要素について記載します。

## HTMLテンプレート
- `templates/index.html` のみ
  - シングルページ構成
  - SVG円形進捗バー、タイマー表示、開始/リセットボタン、進捗統計（完了回数・集中時間）

## JavaScript
- `static/js/timer.js`：
  - タイマーの状態管理、UI更新、イベントハンドラ（開始・リセットボタン）
  - 完了回数・集中時間のカウント
- `static/js/timer_logic.js`：
  - タイマー進捗計算・表示ロジック（`calcDashOffset`、`formatTime`、`getStatusLabel`、`formatFocusTime`）
  - Node.js/Jest用エクスポート対応

## CSS
- `static/css/style.css`：
  - ウィンドウ風デザイン、SVG進捗リング、ボタン、統計表示などのスタイル

## UI要素
- タイトルバー（ウィンドウ風）
- SVG進捗リング（残り時間を可視化）
- タイマー表示（mm:ss）
- 状態ラベル（作業中/停止中/完了）
- 開始・リセットボタン
- 完了回数・集中時間の統計

## アクセシビリティ
- タイマー表示に `aria-live` 属性
- ボタン・ラベルに明確なID

---

# 注意
- すべての状態管理・ロジックはフロントエンドで完結
- バックエンドはHTMLテンプレートの返却のみ
