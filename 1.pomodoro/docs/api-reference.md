# APIリファレンス

このドキュメントは、実装済みのポモドーロタイマーWebアプリのREST API仕様を記載します。

## エンドポイント一覧

現状、Flaskアプリは以下のエンドポイントのみを提供しています。

### ルート
- `GET /`
  - 概要: タイマーUIを含むHTMLページを返します。
  - レスポンス: `text/html` (テンプレート: `index.html`)

> ※現時点でAPIとしてのJSONエンドポイントやデータ操作用エンドポイントは実装されていません。

---

# フロントエンドの主要なJS関数

- `timer.js` および `timer_logic.js` で定義されている関数:
  - `updateDisplay()`：タイマー表示・進捗・状態ラベル・ボタン・統計情報を更新
  - `tick()`：1秒ごとに残り時間を減らし、完了時にカウント・集中時間を加算
  - `calcDashOffset(timeLeft, totalTime)`：SVG進捗バーのstroke-dashoffset計算
  - `formatTime(seconds)`：mm:ss形式に変換
  - `getStatusLabel(isRunning, timeLeft)`：状態ラベル（作業中/停止中/完了）を返す
  - `formatFocusTime(minutes)`：集中時間を「xx分」「x時間xx分」形式で返す

---

# UI操作

- 「開始」ボタン: タイマー開始/一時停止を切り替え
- 「リセット」ボタン: タイマーを初期値に戻す
- 進捗リング: SVGで残り時間を可視化
- 完了回数・集中時間: 画面下部に統計表示
